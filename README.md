# File-Encryptor
### A file encryption program that allows a user to encrypt/decrypt a file whenever they want. User control and access was designated by a password of their choice. The UI theme took on an interesting style during development

Ways to start program:
<ul>
  <li>cd into 'File Encryptor' and run 'python gui_main.py'</li>
  <li>cd into 'File Encryptor' and run 'python main.py {encrypt,decrypt} (input_path)'</li>
</ul>

If you are having issues running, 'python main.py -h' should pop up with some information regarding and questions.
This will also show more options avaiable in the CLI, including 'delete' and specific output paths.
